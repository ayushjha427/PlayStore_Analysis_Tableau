# Google Play Store Data Analysis (Tableau Project)

## Project Overview

This project focuses on analyzing Google Play Store application data to understand the factors that influence app performance, user engagement, and overall success. The analysis was performed using Tableau to transform raw data into meaningful insights through interactive visualizations and dashboards.

The objective of this project is to identify patterns related to app ratings, installs, pricing models, content ratings, and category performance to support data-driven decision-making.

---

## Dataset Information

The dataset contains information about various applications available on the Google Play Store, including:

* App Name
* Category
* Rating
* Reviews
* Installs
* Size
* Type (Free/Paid)
* Price
* Content Rating
* Genres
* Last Updated
* Android Version

Dataset Source: Kaggle - Google Play Store Apps Dataset

---

## Data Preparation

Before creating visualizations, the dataset was cleaned and transformed to ensure accurate analysis.

### Data Cleaning Performed

* Removed duplicate records
* Handled missing values
* Converted Installs column into numeric format
* Converted Price column into numeric values
* Standardized categorical values
* Formatted date fields properly
* Prepared fields for visualization and KPI calculations

---

## Key Insights

* Free apps dominate the Google Play Store ecosystem.
* Communication, Social, and Gaming categories attract the highest number of installs.
* Higher-rated applications generally achieve stronger user engagement.
* Most applications are designed for the "Everyone" content category.
* User ratings play a significant role in overall app success and visibility.

---

## Recommendations

* Focus on maintaining app ratings above 4.0 to improve user trust and engagement.
* Prioritize development in categories with high user demand.
* Adopt freemium or ad-supported business models to maximize reach.
* Regular updates and performance improvements can help sustain user satisfaction.
* Analyze successful categories to identify features that drive adoption.
